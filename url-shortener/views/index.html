URL Shortener Microservice

URL Shortener

Enter a valid URL and you'll get it's shortened version as a response:

{"original_url":"https://www.google.com","short_url":1}

From that endpoint just add the "short_url" value and you'll be redirected there:

[this_url]/api/shorturl/1 => https://www.google.com/
